第五章 gitbook的部署

五、gitbook的部署

这里我们选择将电子书部署到github(需要github账号)

首先在当前目录下创建git目录

git init

然后打开https://github.com/，新建一个项目，根据它的提示将文件上传仓库

git remote add origin git@github.com:xxx/xxx.git
git add .
git commit -m 'xxx'
git push -u origin master

接着新建一个分支保存源码

git checkout -b gh-pages

然后将除了_book全部删掉，而_book里的东西到根文件夹下

git add .
git commit -m 'xxx'
git push origin gh-pages

然后就可以到github上选择gh-pages，点击Settings-General-GitHub Pages中的链接，查看自己的网站地址拉~~
