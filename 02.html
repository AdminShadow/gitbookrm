第二章 gitbook的安装

二、GitBook的安装

gitbook是一个基于Node.js的命令行工具，所以要先安装Node.js(下载地址：https://nodejs.org/en/，找到对应平台的版本安装即可)。Node.js都会默认安装npm（node包管理工具），所以不需要单独安装npm，本机使用的版本为

node.js版本：v12.20.1 npm版本：6.14.10

npm -v
# 6.14.10

打开命令行，执行以下命令安装GitBook:

npm install -g gitbook-cli

安装好后可以通过以下命令查看是否安装成功

gitbook -V
# 如果出现以下内容，说明已成功安装
CLI version: 2.3.2
GitBook version: 3.2.3
